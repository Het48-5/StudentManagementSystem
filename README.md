# Student Management System

This is a simple **Student Management System** built with **Python**.

## Features
- Add student
- View students
- Search student
- Delete student

## How to Run
1. Make sure you have Python installed.
2. Open the terminal in this project folder.
3. Run the program:
### Menu Options

When you run `main.py`, you will see a menu like this:


- **1 Add** → Add a new student by entering name, roll number, and course.  
- **2 View** → View all students in the system.  
- **3 Search** → Search for a student using roll number.  
- **4 Delete** → Delete a student using roll number.  
- **5 Exit** → Close the program.
